#  HR Analytics Dashboard | İnsan Kaynakları Analitik Raporu

Bu proje, İnsan Kaynakları verilerinin Excel üzerinde tamamen Pivot Table, Pivot Chart ve Slicer yapıları kullanılarak analiz edildiği profesyonel bir dashboard çalışmasını içerir.  
Amacım, IK süreçlerinin hızlı ve dinamik şekilde izlenebileceği, formülsüz fakat güçlü bir analiz modelini pratik olarak göstermektir.

This project presents a professional HR analytics dashboard created fully with **Excel Pivot Tables, Pivot Charts, and Slicers** — without using any formulas. The goal is to design a clear, dynamic, and business-focused reporting structure suitable for HR and data-driven decision-making.

---

##  Project Purpose | Projenin Amacı

Bu çalışma, özellikle **IK Raporlama**, **Veri Analizi** ve **Dashboard tasarımı** alanlarında güçlü bir örnek oluşturmak için hazırlandı

- İnsan kaynakları verilerini anlamlandırmak  
- Etkileşimli bir dashboard tasarlamak  
- Veri görselleştirme prensiplerini uygulamak  
- Formül kullanmadan, Pivot mimarisi ile profesyonel bir raporlama oluşturmak  

The purpose of this project is to demonstrate strong skills in:

- HR data interpretation  
- Interactive dashboard design  
- Pivot-based analytical modeling  
- Professional reporting with no formulas needed

##  Dashboard Features | Dashboard Özellikleri

###  Pivot Tabanlı Analizler
Dashboard tamamen Pivot yapısı üzerine kuruldu:

- Departman bazlı çalışan dağılımı  
- Cinsiyet analizi  
- Yaş segmentasyonu  
- Eğitim seviyeleri  
- Çalışma tipi kırılımları  
- İşe giriş yılı / kıdem analizi  

###  Görsel Pivot Chart Dashboard
- Bar, pie, column ve donut grafikler  
- Temiz & sade bir tasarım anlayışı  
- Renk uyumlu tema  
- Yönetici sunumlarına uygun yapı  

###  Slicer ile Etkileşimli Filtreleme
Dashboard kullanıcısı:

- Departman  
- Cinsiyet  
- Eğitim  
- Çalışma tipi  

gibi kriterlere göre **tek tıkla dinamik analiz** yapabilir.

##  How to Use | Nasıl Kullanılır

1. Dosyayı açın  
2. Dashboard sayfasına girin  
3. Sağdaki slicer'ları kullanarak filtreleme yapın  
4. Veriyi güncellemek için:  
   **Pivot Table → Sağ Tık → Refresh**
